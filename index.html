Introduction au Hacking Éthique avec Python

Le hacking éthique est une discipline cruciale dans le domaine de la cybersécurité. Python, grâce à sa simplicité et sa puissance, est devenu l'un des langages les plus utilisés dans ce domaine. Dans cette formation avancée, nous allons explorer en profondeur les bibliothèques Python essentielles pour le hacking éthique.

Scapy: Manipulation de Paquets Réseau

Scapy est une bibliothèque puissante pour la création, la manipulation et l'analyse de paquets réseau. Elle permet aux hackers éthiques de simuler des attaques, d'analyser le trafic et de tester la sécurité des réseaux.

from scapy.all import * # Envoi d'un paquet ICMP (ping) ping = IP(dst="8.8.8.8")/ICMP() response = sr1(ping, timeout=2, verbose=False) if response: print(f"Réponse reçue de {response.src}") else: print("Aucune réponse reçue")

Requests: Interactions HTTP Avancées

Requests simplifie grandement les requêtes HTTP, ce qui est essentiel pour le web scraping, les tests de sécurité web et l'interaction avec les API.

import requests # Effectuer une requête GET response = requests.get('https://api.github.com/users/octocat') if response.status_code == 200: data = response.json() print(f"Nom d'utilisateur: {data['login']}") print(f"Bio: {data['bio']}") else: print(f"Erreur: {response.status_code}")

BeautifulSoup: Parsing HTML et XML

BeautifulSoup est un outil puissant pour l'extraction de données à partir de pages HTML et XML. Elle est souvent utilisée en conjonction avec Requests pour le web scraping.

import requests from bs4 import BeautifulSoup # Récupérer le contenu d'une page web url = 'https://news.ycombinator.com/' response = requests.get(url) soup = BeautifulSoup(response.text, 'html.parser') # Extraire les titres des articles for title in soup.find_all('a', class_='storylink'): print(title.text)

Paramiko: Automatisation SSH

Paramiko permet d'établir des connexions SSH et d'exécuter des commandes à distance. C'est un outil précieux pour l'automatisation des tâches de sécurité et les tests de pénétration.

import paramiko # Établir une connexion SSH ssh = paramiko.SSHClient() ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy()) ssh.connect('example.com', username='user', password='password') # Exécuter une commande stdin, stdout, stderr = ssh.exec_command('ls -l') print(stdout.read().decode()) ssh.close()
